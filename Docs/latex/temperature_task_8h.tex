Function to be given to the program\textquotesingle{}s scheduler to handle monitoring thermal sensors and displaying temperature in the L\+CD and L\+ED displays. This task handles all relevant setup and shutdown procedures as well as calling for regular updates. The task sleeps for 1 second between each update in order to avoid flickering in the output and give the user time to read the displayed information.
