Calculates the speed and the distance travelled of chico. It calculates the speed of both the right and left wheel. Then does the average to return the speed. Then it calculates the distance travelled using the left wheel speed and the period.

Parameters
periodRepresents the period of that the calculation is done.
returnSpeedA pointer to a float in which the speed will be stored.
returnDistanceTravelledA pointer to a float in which the distance travelled will be stored.
